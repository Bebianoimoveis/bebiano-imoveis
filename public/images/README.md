# Imagens do site

Pasta para imagens fixas do site (logo, favicon, banners institucionais) —
diferente das fotos de imóveis, que são enviadas pelo painel administrativo
e armazenadas automaticamente no Cloudinary.

Arquivos colocados aqui ficam acessíveis publicamente em `/images/<nome-do-arquivo>`.

Exemplo: `public/images/logo.png` fica disponível em `https://.../images/logo.png`.
